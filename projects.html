Meal Plan Calendar

  • This application uses the Full Calendar from FullCalendar.io, and was also created using the ASP.NET framework.
  • This website uses a user registration and login to personalize the users calendar.
  • Events can be added or removed from the calendar. These events are varied menu items that are stored in the MSSQL database.
  • Local Storage is used in the browser to save the events that have been added to the calendar.
  • There is a Menu Items page which displays the menu item details.
  • Additionally, there is a search feature which will allow the user to search for terms that will help them query the menu items, such as "chicken", "beef", etc.
  • Repository: https://github.com/JYnfante/MealPlanCalendar
